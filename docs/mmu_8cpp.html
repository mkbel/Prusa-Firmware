◆ can_extrude()

bool can_extrude ( )

Do load steps only if temperature is higher then min. temp for safe extrusion and idler sensor present. Otherwise "cold extrusion prevented" would be send to serial line periodically and watchdog reset will be triggered by lack of keep_alive processing.

Return values
truetemperature is high enough to move extruder
falsetemperature is not high enough to move extruder, turned off E-stepper to prevent over-heating and allow filament pull-out if necessary

◆ mmu_command()

void mmu_command ( MmuCmd  cmd)

Call manage_response() after enqueuing to process command. If T command is enqueued, it disables current for extruder motor if TMC2130 driver present. If T or L command is enqueued, it marks filament loaded in AutoDeplete module.
