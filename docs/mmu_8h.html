◆ mmu_command()

void mmu_command ( MmuCmd  cmd)

Call manage_response() after enqueuing to process command. If T command is enqueued, it disables current for extruder motor if TMC2130 driver present. If T or L command is enqueued, it marks filament loaded in AutoDeplete module.
